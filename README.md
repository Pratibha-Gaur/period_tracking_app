#  Period Tracker App (Flutter)

A simple and elegant **Period Tracker** built using **Flutter & Dart**, designed to help users track their menstrual cycle, predict upcoming periods, and view important cycle phases in a clean calendar interface.

---

##  Features

* Basic Login & Registration (local)
* Interactive Calendar View
* Highlights Last Period Days
* Predicts Next Period Cycle
* Shows Ovulation Window
* Local Data Storage using SharedPreferences
* Clean UI with Pink Theme
* Works on Web & Desktop

---

## Tech Stack

* **Flutter**
* **Dart**
* **SharedPreferences**
* **Table Calendar Package**

##  Getting Started

### 1. Clone the repository

```
git clone https://github.com/your-username/period-tracker.git
cd period-tracker
```

---

### 2. Install dependencies

```
flutter pub get
```

---

### 3. Run the app

####  Run on Chrome (Web)

```
flutter run -d chrome
```

####  Run as Desktop App (Windows)

```
flutter run -d windows
```

---

##  Important Notes

* Avoid running the project inside **OneDrive/Desktop** to prevent permission issues.
* Recommended path:

```
C:\flutter_projects\
```
